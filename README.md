# 📝 ToDo App

A simple and user-friendly ToDo application that helps users manage their daily tasks efficiently.

## 🚀 Features
- ➕ Add new tasks
- ✅ Mark tasks as completed
- ❌ Delete tasks
- 💾 Data saved in local storage
- 📱 Responsive design (works on mobile & desktop)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
